# Sample Inputs & Outputs

All outputs below were generated by running the deterministic rule-based router directly (no API key required), so they are exactly reproducible:

```bash
python -c "from agent.rule_based_router import run_agent_rule_based as run; print(run('YOUR QUESTION HERE'))"
```

With a Gemini API key configured, `run_agent()` produces similar answers but phrased by the LLM itself — wording varies slightly run to run since it's genuine model generation, but the underlying tool calls and facts are identical. An example of that LLM-generated phrasing is included at the bottom of this file for comparison.
